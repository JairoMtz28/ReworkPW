# ReworkPW

Este proyecto es una aplicación desarrollada en **Visual Basic .NET** con **Windows Forms**, diseñada para **registrar en la base de datos las piezas que, por diversos motivos, son enviadas a retrabajo dentro de una línea de producción**.

## Descripción

El programa original ya existía como herramienta de apoyo en planta; sin embargo, se realizó una **actualización y mejora integral del sistema**, con el objetivo de hacerlo más robusto, automatizado y adaptable a las nuevas necesidades de producción.  

La aplicación permite **escanear, registrar y gestionar retrabajos de piezas**, garantizando un control preciso de cuántas veces una pieza puede ser retrabajada y enviando automáticamente a **scrap** aquellas que excedan el límite permitido.

## Nuevas funcionalidades implementadas

- **Soporte para una nueva variante de producto** dentro de la misma línea de producción.  
- **Control de número máximo de retrabajos**: al superar el límite establecido, la pieza se marca y envía directamente a **scrap**.  
- **Registro de fallas por lotes completos**, mejorando la trazabilidad de los defectos.  
- **Optimización de la interfaz y flujo de trabajo**, facilitando el uso por parte del personal de producción.  
- **Integración con la base de datos MySQL** para almacenar los registros de manera segura y centralizada.

## Tecnologías utilizadas

- **Visual Basic .NET (Windows Forms)**  
- **MySQL**  
- **.NET Framework**  

## Objetivo

Mejorar el control y seguimiento de piezas retrabajadas en la línea de producción, reduciendo errores manuales, optimizando el tiempo de registro y asegurando que cada pieza cumpla con los límites definidos antes de considerarse **scrap**.
